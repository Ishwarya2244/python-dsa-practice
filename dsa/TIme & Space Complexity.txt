# Time and Space Complexity

## 📘 What is Time Complexity?

Time Complexity tells us how the running time of an algorithm grows as the input size increases.

It is represented using **Big O Notation**.

---

## 📘 What is Space Complexity?

Space Complexity tells us how much **extra memory** an algorithm uses as the input size increases.

It also uses **Big O Notation**.

---

# Common Time Complexities

## 1. O(1) - Constant Time

The work does not change even if the input size increases.

### Example

```python
print("Hello")
```

**Time Complexity:** O(1)

---

## 2. O(n) - Linear Time

The algorithm visits every element exactly once.

### Example

```python
for x in numbers:
    print(x)
```

**Time Complexity:** O(n)

---

## 3. O(n²) - Quadratic Time

A loop inside another loop.

### Example

```python
for i in range(n):
    for j in range(n):
        print(i, j)
```

**Time Complexity:** O(n²)

---

# Common Space Complexities

## O(1)

Uses only a fixed number of extra variables.

### Example

```python
largest = numbers[0]

for x in numbers:
    if x > largest:
        largest = x
```

Extra variables:
- largest
- x

**Space Complexity:** O(1)

---

## O(n)

Creates a new data structure that grows with the input size.

### Example

```python
copy = []

for x in numbers:
    copy.append(x)
```

The new list grows as the input grows.

**Space Complexity:** O(n)

---

# Quick Revision Table

| Pattern | Time | Space |
|---------|------|-------|
| print() | O(1) | O(1) |
| One loop | O(n) | O(1) |
| Nested loops | O(n²) | O(1) |
| New list | O(n) | O(n) |

---

# Easy Memory Tricks

✅ No loop → O(1)

✅ One loop → O(n)

✅ Nested loops → O(n²)

✅ Creating a new list → Space O(n)

---

# Interview Questions

### 1. What is Time Complexity?

Time Complexity measures how the running time of an algorithm grows as the input size increases.

---

### 2. What is Space Complexity?

Space Complexity measures how much extra memory an algorithm uses.

---

### 3. Why is one loop O(n)?

Because every element is visited exactly once.

---

### 4. Why are nested loops O(n²)?

Because for each element, the inner loop visits all elements again.

---

### 5. Why is creating a new list Space O(n)?

Because the extra memory increases as the input size increases.

---

# Key Points

- O(1) → Constant Time
- O(n) → Linear Time
- O(n²) → Quadratic Time
- O(1) Space → Fixed extra memory
- O(n) Space → Extra memory grows with input
