SQL DAY 5 - MIN() and MAX()

MAX()

Used to find the highest value in a column.

Example:

SELECT MAX(marks) FROM students;

Output:
95

--------------------------------

MIN()

Used to find the lowest value in a column.

Example:

SELECT MIN(marks) FROM students;

Output:
60

--------------------------------

Memory Trick

MAX() = Maximum Value

MIN() = Minimum Value

--------------------------------

Connection with DSA

Largest Number  -> MAX()

Smallest Number -> MIN()
